Introducción

JavaScript es un lenguaje de programación interpretado, orientado a objetos, y utilizado principalmente para desarrollo web.

Permite crear sitios web dinámicos e interactivos, ejecutándose del lado del cliente (navegador).

console.log("¡Hola mundo!");
Diferencia con Java

Aunque comparten nombre, JavaScript y Java son lenguajes completamente distintos.

Java es un lenguaje compilado, mientras que JavaScript es interpretado y se ejecuta en el navegador.

  • Java necesita máquina virtual (JVM).
  • JavaScript corre directamente en navegadores.
// JavaScript function saludar() { alert("Hola"); }
Variables

Las variables son contenedores para almacenar datos. En JavaScript se pueden declarar con var, let y const.

El uso de let y const es preferido en la mayoría de los casos modernos.

let nombre = "Juan";
  • var: declaración antigua.
  • let: para variables mutables con alcance de bloque.
  • const: para constantes.
Cómo declarar variables

Se puede declarar una variable usando var, let o const.

La sintaxis es simple: palabra clave + nombre + (opcionalmente) valor.

const PI = 3.14; let edad = 25; var ciudad = "Buenos Aires";
Alcance de variables

El alcance define desde qué lugar del código se puede acceder a una variable.

var tiene alcance de función, mientras que let y const tienen alcance de bloque.

function ejemplo() { if (true) { let x = 5; console.log(x); } // console.log(x); // Error: x no está definida }
Variables globales

Las variables definidas fuera de cualquier función tienen alcance global.

Esto significa que pueden ser accedidas desde cualquier parte del script.

let global = "Hola mundo"; function mostrar() { console.log(global); }
Constantes

Una constante es una variable cuyo valor no puede cambiar.

Se declara con const y debe inicializarse al momento de declararla.

const NOMBRE = "Messi";
  • No se puede reasignar.
  • Tiene alcance de bloque.
Tipos de datos

JavaScript soporta varios tipos de datos:

  • String
  • Number
  • Boolean
  • Object
  • Array
  • Null
  • Undefined
  • Symbol
let nombre = "Lionel"; // String let edad = 36; // Number let activo = true; // Boolean
Condicional if...else

Esta estructura permite ejecutar diferentes bloques de código según una condición.

if (edad >= 18) { console.log("Es mayor de edad"); } else { console.log("Es menor de edad"); }
Condicional while

El bucle while ejecuta un bloque de código mientras la condición sea verdadera.

let i = 0; while (i < 5) { console.log(i); i++; }
Declaración de funciones

Una función permite agrupar instrucciones reutilizables.

Se declara con la palabra clave function, seguida de un nombre y paréntesis.

function saludar(nombre) { return "Hola, " + nombre; } console.log(saludar("Leo"));
